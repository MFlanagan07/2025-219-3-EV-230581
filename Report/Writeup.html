Final Idea:

From Brief,
“Data is vitally important today because it helps people make smart decisions and come up
with new ideas”
While searching for datasets to do my project on I found a dataset containing information on
Pokémon Generation 1 - 6, Pokémon contains a large number of different ways to approach
the game depending on the team build you wish to make to beat the game, and each
generation containing a different prominent theme and type advantage would make for an excellent dataset to use for predictions.
The Dataset had all of the relevant information I would need i.e. Information on each
Pokémon's stats, their types, their Pokedex number and generation, it is also relevant today
as Pokémon games still to this day have games based on generations 1-6 coming out annually.
Therefore, I chose this dataset to use in my project.

Plan and Design

I selected this dataset on Pokemon as Pokemon is a franchise I have loved my entire life
and I consider my knowledge on the Game to be quite sufficient to effectively
make an interactive information system on it.
As Pokemon is a globally known and enjoyed franchise the target audience is as great as
could be (Target age range: Teenagers & Adult fans of Pokemon as they are most likely to be looking for an information system like this),
both kids and adults alike play pokemon and can benefit from the information
on my website, my goal SOCIALLY for the project is to make a comprehensive, educational information system
that informs Pokemon players of hidden factors in games in the hopes of making every user who looks at my website
just a little bit better at Pokemon. I'm taking an AGILE approach to the project so that it has concrete functionality,
I will be approaching the project from the bottom up in phases to deliver an effective information system
My website will help the target audience by visualizing key factors in Pokemon games that
would otherwise not be known without studying the games internal data in a simple
understandable way that even a Child would be able to understand, by extracting data from
the downloaded csv file into comprehensible dictionaries.
